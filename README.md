This repository contains multiple **Python practice programs** that I created while learning Python programming.
These programs cover basic to intermediate Python concepts such as functions, loops, dictionaries, object-oriented programming, and simple mini-projects.

## Topics Covered

* Basic Python Syntax
* Conditional Statements
* Loops (for, while)
* Functions
* Dictionaries and Lists
* Object Oriented Programming (OOP)
* User Input Handling
* Simple Console Applications

## Example Projects

### Student Management System

* Methods

## How to Run the Programs

1. Install Python on your system.
2. Clone this repository:

```
git clone https://github.com/your-username/python.git
```

3. Open the project folder.
4. Run any Python file:

```
python filename.py
```
